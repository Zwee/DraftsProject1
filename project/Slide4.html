HTML Attributes

|- A Bit About HTML Attributes -|

Attributes provide additional info about the element.

Attributes come in pairs, name/value. Think: name/Roger, hair/Brown, age/27.
Attributes are always mentioned in the opening tag.


Examples of Attributes:

  • <a>, </a> = anchor tag, used for making links
  • <a href...>, </a> = href tag (think Hyperlink Reference) (the 'href' is not repeated in the closing tag)
  • <img> = image tag. This tag does not need to be closed. It is used to add images.
